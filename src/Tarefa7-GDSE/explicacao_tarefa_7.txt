A DTD externa define a estrutura de um documento XML que contém informações de contato de pessoas em um banco de dados. O elemento raiz db deve conter um ou mais elementos pessoa, cada um dos quais deve conter um elemento nome, um elemento idade, um ou mais elementos email e um elemento telefone opcional. Os elementos nome, idade, email e telefone devem conter apenas dados de texto (PCDATA). Em resumo, essa DTD garante que o documento XML esteja bem estruturado e contenha informações de contato válidas para cada pessoa.

Assim para utilizamos a DTD no arquivo xml adicionamos o campo <!DOCTYPE db SYSTEM "DTD.dtd"> ao inicio do arquivo XML logo o XML é definido usando a estrutura da DTD externa. Quando o documento XML é validado contra a DTD externa, o parser XML verifica se o XML está bem formado e se todos os elementos e atributos estão corretos.
